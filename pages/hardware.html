Quick Reference

Key Board Features

  • 8-core Propeller P8X32A microcontroller - Multi-core processing capability
  • Integrated solderless breadboard - Easy prototyping
  • Wi-Fi module socket - Wireless connectivity options
  • Multiple power options - USB or external power supply
  • Six servo headers - Easy servo motor connections
  • Analog/Digital converters - Sensor interfacing
  • MicroSD card slot - Data logging and file storage
