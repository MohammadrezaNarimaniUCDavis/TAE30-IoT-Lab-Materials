Lab 2: Sensors Basics

Working with temperature, humidity, and light sensors

Required Materials

  • Arduino Uno or similar microcontroller
  • Breadboard and jumper wires
  • DHT11 or DHT22 temperature and humidity sensor
  • Photoresistor (LDR)
  • 10kΩ resistor
  • USB cable

Lab Objectives

By the end of this lab, you will be able to:

  • Understand how different sensors work
  • Connect temperature, humidity, and light sensors to Arduino
  • Read and process sensor data
  • Display sensor readings
  • Implement basic threshold detection

Step-by-Step Tutorial

1. Understanding Sensors

Sensors are devices that detect and measure physical properties from the environment and convert them into electrical signals.

  • DHT11/DHT22: Digital temperature and humidity sensor
  • Photoresistor (LDR): Light-dependent resistor that changes resistance based on light intensity

2. Connecting the DHT Sensor

Connect the DHT11 or DHT22 sensor to your Arduino:

  1. Connect VCC pin to 3.3V or 5V on Arduino
  2. Connect GND pin to GND on Arduino
  3. Connect DATA pin to digital pin 2 on Arduino
  4. Add a 10kΩ pull-up resistor between VCC and DATA pins

3. Reading Temperature and Humidity

Create a program to read temperature and humidity data:

  1. In BlocklyProp Solo, create a new project
  2. Add the DHT sensor library blocks
  3. Initialize the DHT sensor on pin 2
  4. Read temperature and humidity values in a loop
  5. Display the values using serial communication

4. Adding Light Sensor

Connect and program the photoresistor:

  1. Connect one leg of the photoresistor to analog pin A0
  2. Connect the other leg to 5V
  3. Connect a 10kΩ resistor from A0 to GND (voltage divider)
  4. Read the analog value and convert to light intensity

5. Implementing Threshold Detection

Add logic to trigger actions based on sensor readings:

  1. Set temperature thresholds (e.g., too hot > 25°C)
  2. Set light thresholds (e.g., dark < 300)
  3. Use conditional blocks to trigger LED indicators
  4. Test different environmental conditions
