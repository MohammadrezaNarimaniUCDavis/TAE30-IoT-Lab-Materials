Lab 4: Data Collection

Collect and store sensor data locally

Required Materials

  • Arduino Uno with SD card shield
  • MicroSD card
  • Temperature and humidity sensor (DHT22)
  • Light sensor (photoresistor)
  • Real-time clock module (RTC)
  • Breadboard and jumper wires

Lab Objectives

By the end of this lab, you will be able to:

  • Set up data logging with SD card
  • Implement timestamping with RTC
  • Create CSV data files
  • Handle data storage efficiently
  • Implement data collection intervals
