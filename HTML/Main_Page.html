TAE 30: Communications and Computing Technology

Course Introduction

Welcome to TAE 30: Communications and Computing Technology. This course introduces students to the Internet of Things (IoT) concepts, technologies, and applications with a focus on hands-on learning. Throughout this course, you will learn how to build IoT systems using various sensors, actuators, and computing platforms.

By the end of this course, you will be able to design, build, and program IoT devices that can collect data, communicate wirelessly, and interact with cloud services.

Lab 1: Introduction to IoT

Getting started with IoT concepts and development environment

Required Materials

  • Microcontroller board (Arduino Uno or similar)
  • Breadboard
  • Jumper wires
  • USB cable
  • LED (any color)
  • 220Ω resistor

Lab Objectives

By the end of this lab, you will be able to:

  • Understand basic IoT concepts and terminology
  • Set up your development environment with BlocklyProp Solo
  • Create a simple program to blink an LED
  • Upload code to your microcontroller
  • Modify parameters to change LED behavior

Step-by-Step Tutorial

1. Setting Up BlocklyProp Solo

BlocklyProp Solo is a visual programming environment that allows you to create programs using blocks that snap together.

  1. Open your web browser and navigate to https://solo.parallax.com/
  2. Click on "Create New Project"
  3. Select "Arduino" as your board type
  4. Name your project "Lab1_Blink_LED"

BlocklyProp Solo Interface

2. Building the Circuit

Let's build a simple circuit with an LED:

  1. Connect the longer leg (anode) of the LED to digital pin 13 on the Arduino
  2. Connect the shorter leg (cathode) of the LED to one end of the 220Ω resistor
  3. Connect the other end of the resistor to the GND (ground) pin on the Arduino

LED Circuit Diagram

3. Creating the Blink Program

Now let's create a program to blink the LED:

  1. In BlocklyProp Solo, find the "Control" category in the toolbox
  2. Drag the "Setup" block to the workspace
  3. From the "Pins" category, drag the "Set pin [?] to [OUTPUT]" block inside the Setup block
  4. Change the pin number to 13
  5. From the "Control" category, drag the "Forever" loop block below the Setup block
  6. Inside the Forever loop, add these blocks in sequence:
    • From "Pins" category: "Set pin [13] to [HIGH]"
    • From "Control" category: "Pause (ms) [500]"
    • From "Pins" category: "Set pin [13] to [LOW]"
    • From "Control" category: "Pause (ms) [500]"

BlocklyProp Blink Program

4. Uploading and Testing

Now let's upload the program to the Arduino:

  1. Connect your Arduino to your computer using the USB cable
  2. In BlocklyProp Solo, click the "Compile" button
  3. Once compilation is complete, click "Load to Board"
  4. Select your Arduino from the list of available devices
  5. Click "Upload" to send the program to your Arduino
  6. After uploading, the LED should start blinking - turning on for 0.5 seconds and off for 0.5 seconds

5. Modifying the Program

Let's modify the program to change the blinking pattern:

  1. Change the first "Pause" block from 500ms to 200ms (faster on time)
  2. Change the second "Pause" block from 500ms to 800ms (longer off time)
  3. Compile and upload the program again
  4. Observe how the blinking pattern has changed

Try experimenting with different timing values to create your own blinking patterns!

TAE 30: Communications and Computing Technology

Course Description

This course introduces students to the Internet of Things (IoT) concepts, technologies, and applications with a focus on hands-on learning. Students will learn how to build IoT systems using various sensors, actuators, and computing platforms.

Learning Objectives

  • Understand fundamental IoT concepts and architectures
  • Work with various sensors and actuators
  • Program microcontrollers using visual programming tools
  • Collect, process, and visualize data from IoT devices
  • Implement wireless communication between devices
  • Integrate IoT devices with cloud services
  • Apply IoT solutions to real-world problems

Frequently Asked Questions

Common questions about the course and labs

Do I need prior programming experience?

No prior programming experience is required. We'll be using BlocklyProp Solo, which is a visual programming environment that makes it easy to get started with coding.

What materials do I need for this course?

You'll need an Arduino Uno (or compatible) board, a breadboard, jumper wires, and various sensors and actuators. A complete list of required materials will be provided at the beginning of the course.

Can I work on the labs using my own computer?

Yes, you can work on the labs using your own computer. You'll need to install the required software (BlocklyProp Solo and Arduino IDE) and have a USB port to connect your Arduino board.

How are the labs graded?

Labs are graded based on completion, functionality, and a short lab report. Each lab is worth 10% of your final grade, with the final project worth 20%.

What if I miss a lab session?

If you miss a lab session, you can make it up during office hours or by following the online lab materials. Please contact the instructor or TA as soon as possible to make arrangements.

Can I collaborate with other students on the labs?

You are encouraged to discuss concepts and troubleshoot with your classmates, but each student must submit their own work. Copying code or lab reports is considered academic dishonesty.
